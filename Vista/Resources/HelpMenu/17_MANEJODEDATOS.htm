MANEJO DE DATOS

BACK-UP

1.      En la pantalla principal debemos seleccionar sobre la barra de herramientas la opción Herramientas>Base de Datos>Realizar Back-Up

2.      Se mostrará una ventana donde debe seleccionar el destino del archivo de back-up.

3.      Una ventana de informe se visualizará cuando finalice el proceso de back-up.
