RESTAURAR BACK-UP

1.      En la pantalla principal debemos seleccionar sobre la barra de herramientas la opción Herramientas>Base de Datos>Restaurar Base de Datos

2.      Se mostrará una ventana donde debe seleccionar el archivo .bak que contiene el back-up a restaurar.

3.      Una ventana de informe se visualizará cuando finalice el proceso de restauración.
