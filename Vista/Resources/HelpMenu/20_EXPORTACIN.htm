EXPORTACIÓN

 

1.      En la pantalla principal debemos seleccionar sobre la barra de herramientas la opción Herramientas>Exportar Datos y seleccionar el modelo a exportar.

2.      Se mostrará una ventana donde debe seleccionar el destino del archivo de exportación.

3.      Una ventana de informe se visualizará cuando finalice el proceso de exportación.
